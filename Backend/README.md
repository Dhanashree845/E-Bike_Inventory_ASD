This is the backend server of the Cloud-Based Inventory Management System built using Node.js and Express.js. It provides RESTful APIs for managing e-bike inventory, authentication, and dashboard analytics.

The backend connects to MongoDB Atlas (or a local MongoDB instance) for data storage and implements JWT-based authentication for secure access.

## Overview

The backend handles:

- User registration and login
- Password hashing
- JWT token generation
- Role-based authorization (ADMIN / STAFF)
- CRUD operations on e-bike inventory
- Customer and supplier management
- Purchase and sales tracking
- Inventory and transaction reports
- Image upload handling
- Dashboard statistics APIs
- Database interaction using Mongoose

## Description

This project is a Cloud-Based E-Bike Inventory Management System backend developed using Node.js and Express.js. It exposes secure REST APIs to perform CRUD operations on e-bike inventory data with authentication and role-based access control.
